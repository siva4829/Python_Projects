# 🧠 Python Projects Collection

A collection of beginner-to-intermediate Python projects built while learning and experimenting with different concepts such as GUI development, logic building, and problem-solving.

---

## 📂 Projects Included

### 🎮 Games

* **Tic Tac Toe**
  A GUI-based Tic Tac Toe game with:

  * Player vs Player mode
  * Player vs Computer mode
  * Turn tracking and win/draw detection

* **Bingo**
  Simple Bingo game logic implementation.

* **Number Guessing Game**
  Guess the randomly generated number with hints.

---

### 🧰 Utility Applications

* **Digital Clock**
  Displays real-time clock using GUI.

* **Stop Watch**
  A stopwatch with start, stop, and reset functionality.

* **Weather App**
  Fetches and displays weather information.

* **Random Password Generator**
  Generates secure random passwords.

---

### 🗂️ Other Projects

* **Auction House**
  Basic bidding system simulation.

* **To-Do List**
  Task manager to add and track daily tasks.

---

## ⚙️ Technologies Used

* Python 🐍
* PyQt5 (for GUI-based applications)
* Standard Python libraries (random, sys, etc.)

---

## 🚀 How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/Python_Projects.git
   ```

2. Navigate into the project folder:

   ```bash
   cd Python_Projects
   ```

3. Run any project:

   ```bash
   python filename.py
   ```
